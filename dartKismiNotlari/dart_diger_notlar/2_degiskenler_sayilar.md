## Dartta değişkenler
```
void main() {

  int sayi1 = 19;
  print(sayi1);

  double sayi2 = 19.19;
  print(sayi2);

  num numara1 = 1919;
  print(numara1);

  double exponentsSayi1 = 1.919e3;
  print(exponentsSayi1);

  var sayi3 = 19051919;
  print(sayi3);
}
```

##### Diğer programlama dillerinde olan float dartta yoktur.
